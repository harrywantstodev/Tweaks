# Tweaks

Lots of tweaks in one place to learn from

### Prerequisites

What things you need to install the software and how to install them

```
theos
iOS 10.2 Headers
```

### Installing

All of these should just be

```
cd /tweakfolder
```

And 

```
make package
```

or 

```
make package install
```
